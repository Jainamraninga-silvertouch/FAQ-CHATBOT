UPLOAD FOLDER
=============

Place your documents here and restart the server to load them automatically.

SUPPORTED FILE TYPES:
- PDF files (.pdf)
- Word documents (.docx)
- Text files (.txt)
- Excel spreadsheets (.xlsx, .xls)

HOW TO USE:
1. Copy your document files into this folder
2. Restart the server (Ctrl+C and run again)
3. Documents will be automatically indexed
4. Ask questions in the web interface

EXAMPLE:
upload/
  ├── employee_handbook.pdf
  ├── policies.docx
  ├── faq_data.xlsx
  └── README.txt

The chatbot will search across ALL documents to answer your questions.
